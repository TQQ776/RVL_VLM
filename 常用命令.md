## 6. MCP 服务手动测试

先启动完整链路或至少启动 `mcp_server`。

列出服务：

```bash
ros2 service list | grep mcp_server
```

回 home：

```bash
ros2 service call /mcp_server/go_home std_srvs/srv/Trigger {}
```

单轴移动：

```bash
ros2 service call /mcp_server/move_x_cm mcp/srv/MoveAxis "{centimeters: 2.0}"
ros2 service call /mcp_server/move_y_cm mcp/srv/MoveAxis "{centimeters: -2.0}"
ros2 service call /mcp_server/move_z_cm mcp/srv/MoveAxis "{centimeters: 1.0}"
```

夹爪：

```bash
ros2 service call /mcp_server/open_gripper std_srvs/srv/Trigger {}
ros2 service call /mcp_server/close_gripper std_srvs/srv/Trigger {}
```

抓取目标：

```bash
ros2 service call /mcp_server/grab_object mcp/srv/ObjectName "{name: orange}"
ros2 service call /mcp_server/grab_object mcp/srv/ObjectName "{name: apple}"
```

观察并建立场景记忆：

```bash
ros2 service call /mcp_server/call_tool mcp/srv/CallTool \
  "{name: observe_scene, arguments_json: '{\"object_names\": [\"orange\", \"apple\"]}'}"
```

这一步会同时把检测到的水果/箱子/其他清晰独立障碍物发布成 MoveIt collision objects。
如果 RViz 里打开了 PlanningScene/Scene Objects，就会先看到模型，再执行后续抓取或放置动作。

把当前夹着的物体放到已记住物体的相对位置，例如放到苹果右边 5 厘米：

```bash
ros2 service call /mcp_server/place_relative_to_object mcp/srv/PlaceRelative \
  "{held_object_name: orange, reference_object_name: apple, direction: right, distance_cm: 5.0}"
```

完整抓取-放置一条工具调用，例如“把橘子放到苹果右边 5 厘米”：

```bash
ros2 service call /mcp_server/call_tool mcp/srv/CallTool \
  "{name: pick_and_place_relative, arguments_json: '{\"object_name\": \"orange\", \"reference_object_name\": \"apple\", \"direction\": \"right\", \"distance_cm\": 5.0}'}"
```

把当前夹着的物体放进箱子：

```bash
ros2 service call /mcp_server/place_into_container mcp/srv/PlaceIntoContainer \
  "{held_object_name: apple, container_name: box}"
```

完整抓取-放进箱子，例如“把苹果放到箱子里”：

```bash
ros2 service call /mcp_server/call_tool mcp/srv/CallTool \
  "{name: pick_and_place_into_container, arguments_json: '{\"object_name\": \"apple\", \"container_name\": \"box\"}'}"
```

放箱子动作会先沿 z 小幅抬升，再低速 MoveIt 规划到箱口上方。
如果真实机械臂仍然触发保护，可以优先降低这些参数：

```text
tqq/src/mcp/config/mcp_server.yaml
place_max_velocity_scaling
place_max_acceleration_scaling
container_transfer_lift_m
```

查看工具说明：

```bash
ros2 service call /mcp_server/list_tools mcp/srv/ListTools {}
```

用统一工具入口调用：

```bash
ros2 service call /mcp_server/call_tool mcp/srv/CallTool \
  "{name: move_z_cm, arguments_json: '{\"centimeters\": 2.0}'}"
```

```bash
ros2 service call /mcp_server/call_tool mcp/srv/CallTool \
  "{name: grab_object, arguments_json: '{\"name\": \"orange\"}'}"
```

## 15. 常见错误快速处理

### 15.1 `Package 'mcp' not found`

```bash
source ~/TQQ_ws/setup_franka.sh
ros2 pkg prefix mcp
```

如果还找不到：

```bash
cd ~/TQQ_ws/tqq
source ~/TQQ_ws/setup_franka.sh
colcon build --symlink-install
source install/setup.bash
```

### 15.2 `libfranka: Connection to FCI refused`

处理：

```text
去 Franka Desk 启用 FCI
确认机器人可外部控制
确认急停释放
确认 robot_ip 正确
```

检查网络：

```bash
ping 192.168.22.212
```

### 15.3 RealSense `serial_no` 类型错误

YAML 里序列号必须是字符串：

```yaml
serial_no: "327122079035"
```

### 15.4 Qwen / Omni 没有 API Key

```bash
export DASHSCOPE_API_KEY=你的_DashScope_API_Key
echo $DASHSCOPE_API_KEY
```

### 15.5 EconomicGrasp import 失败

```bash
conda deactivate 2>/dev/null || true
source ~/TQQ_ws/setup_franka.sh
/usr/bin/python3 - <<'PY'
import torch
import MinkowskiEngine
import pointnet2
import knn_pytorch

PY
```

### 15.6 LLM 不调用工具

检查服务：

```bash

```

### 15.7 点云预览要不要按 Enter

配置文件：

```text
~/TQQ_ws/tqq/src/economic_grasp_roi/config/roi_economic_grasp.yaml
```

自动执行：

